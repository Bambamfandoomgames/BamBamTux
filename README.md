# BamBamTux

A `bambam` style game rewritten from scratch in `raylib`.

![Screenshot](./Screenshot.png "5 november 2024")

## Run

```sh
./build_and_run.sh
```

Press `Esc` to quit.

## Done so far

- A symbol appear when a printable key is  pressed.
- A `linux` logo appears when other keys are pressed.
